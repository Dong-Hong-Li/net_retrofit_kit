# 注解说明

所有注解定义在 [lib/src/generate/annotations.dart](https://github.com/Dong-Hong-Li/net_retrofit_kit/blob/main/lib/src/generate/annotations.dart)。生成器根据它们生成调用 [NetRequest.requestHttp](https://github.com/Dong-Hong-Li/net_retrofit_kit/blob/main/lib/src/network/net_request.dart)（或具名 [INetClient](https://github.com/Dong-Hong-Li/net_retrofit_kit/blob/main/lib/src/network/inet_client.dart)）的代码。

---

## 类级：`@NetApi`

标在 **abstract** API 类上。

```dart
@NetApi({
  String? client,           // requestHttp 的 clientKey；null 表示 default
  String responseType = 'BaseResponse',
  bool unwrapSuccess = true,
})
abstract class MyApi { ... }
```

| 参数 | 含义 |
|------|------|
| `client` | 传给 `requestHttp(clientKey: ...)` 的 key。若设置（如 `'upload'`），生成代码会使用该 Client；需用 `NetRequest.setClient('upload', yourClient)` 注册。 |
| `responseType` | 解析时使用的响应类型名（如 `BaseResponse`）。 |
| `unwrapSuccess` | 为 `true` 时成功只返回 `response.data`；为 `false` 时返回完整 response。 |

使用自定义 Client 的示例：[example/lib/server/upload_api.dart](https://github.com/Dong-Hong-Li/net_retrofit_kit/blob/main/example/lib/server/upload_api.dart) — `@NetApi(client: 'upload')`。

## 方法级：HTTP 方法 + 路径

| 注解 | HTTP 方法 | 路径 |
|------|-----------|------|
| `@Get(path)` | GET | path（相对 baseUrl） |
| `@Post(path)` | POST | 同上 |
| `@Put(path)` | PUT | 同上 |
| `@Delete(path)` | DELETE | 同上 |

可选：`contentType` 指定请求体类型（如上传用 `ContentType.formData`）。

```dart
@Get('/user/list')
Future<UserList?> getList();

@Post('/user')
Future<User?> create(@Body() UserCreateRequest body);

@Post('/upload', contentType: ContentType.formData)
Future<Result?> upload(@Part('file') File file, @Part('name') String name);
```

[ContentType](https://github.com/Dong-Hong-Li/net_retrofit_kit/blob/main/lib/src/network/net_content_type.dart)：`json`、`formData`、`xWwwFormUrlencoded`。

---

## 参数：body 与 query

| 注解 | 用途 | 生成代码中的用法 |
|------|------|------------------|
| `@Body()` | 请求体 | `body: param` 或 `param.toJson()` |
| `@Query()` | 完整 query map | `queryParameters: param` |
| `@QueryKey(name)` | 单个 query 参数 | 合并到 `queryParameters[name]` |

```dart
@Post('/login')
Future<Auth?> login(@Body() LoginRequest body);

@Get('/search')
Future<List<Item>?> search(@QueryKey('keyword') String q, @QueryKey('page') int page);

@Get('/list')
Future<List<Item>?> list(@Query() Map<String, dynamic> query);
```

---

## 参数：路径与请求头

| 注解 | 用途 | 路径占位 |
|------|------|-----------|
| `@Path(name)` | 路径段 | path 中需包含 `:name`（如 `/user/:id`） |
| `@Header(name)` | 单个请求头 | `headers[name]` |

```dart
@Get('/user/:id')
Future<User?> getUser(@Path('id') String id);

@Get('/resource/:type/:id')
Future<Resource?> getResource(@Path('type') String type, @Path('id') String id);

@Get('/me')
Future<User?> me(@Header('Authorization') String token);
```

---

## 响应解析

| 注解 | 用途 |
|------|------|
| `@DataPath('key')` | 从 `response.data['key']` 解析，而不是从 `response.data`。 |

```dart
@Get('/nested')
@DataPath('result')
Future<NestedModel?> getNested();
```

示例：[example/lib/server/nested_api.dart](https://github.com/Dong-Hong-Li/net_retrofit_kit/blob/main/example/lib/server/nested_api.dart)。

## 多部分表单（文件上传）

使用 `@Post(path, contentType: ContentType.formData)` 与 `@Part(name)`：

```dart
@Post('/post', contentType: ContentType.formData)
Future<Map<String, dynamic>?> upload(
  @Part('file') File file,
  @Part('name') String name,
);
```

生成器会组装 `FormData`；`File` 会转为 `MultipartFile`。示例：[example/lib/server/upload_api.dart](https://github.com/Dong-Hong-Li/net_retrofit_kit/blob/main/example/lib/server/upload_api.dart)。

## 流式 / SSE

| 注解 | 用途 |
|------|------|
| `@StreamResponse()` | 方法返回流；生成器调用 `NetRequest.requestStreamResponse` 并返回响应流（如 SSE、按行）。 |

```dart
@Get('/stream/3')
@StreamResponse()
Future<Stream<String>> getStreamLines({CancelToken? cancelToken});
```

示例：[example/lib/server/demo_server.dart](https://github.com/Dong-Hong-Li/net_retrofit_kit/blob/main/example/lib/server/demo_server.dart) — `getStreamLines`。

---

## 可选：`RequestOptions? options`

方法最后一个参数可为 `RequestOptions? options`。生成器会透传（例如给你的 execute 包装做 loading/错误处理）。类型来自 Dio。

---

## 一览表

| 注解 | 作用范围 | 说明 |
|------|----------|------|
| `@NetApi(...)` | 类 | Client key、响应类型、是否只返回 data |
| `@Get(path)` / `@Post(path)` / `@Put(path)` / `@Delete(path)` | 方法 | HTTP 方法 + 路径 |
| `@Body()` | 参数 | 请求体 |
| `@Query()` | 参数 | 完整 query map |
| `@QueryKey(name)` | 参数 | 单个 query 参数 |
| `@Path(name)` | 参数 | 路径段 `:name` |
| `@Header(name)` | 参数 | 请求头 |
| `@DataPath('key')` | 方法 | 从 `data['key']` 解析 |
| `@Part(name)` | 参数 | 多部分表单（formData） |
| `@StreamResponse()` | 方法 | 流式/SSE 响应 |
