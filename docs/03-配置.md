# 配置说明

全局配置通过 [NetRequest](https://github.com/Dong-Hong-Li/net_retrofit_kit/blob/main/lib/src/network/net_request.dart) 和 [NetOptions](https://github.com/Dong-Hong-Li/net_retrofit_kit/blob/main/lib/src/network/net_options.dart) 完成。

---

## NetOptions

定义在 [lib/src/network/net_options.dart](https://github.com/Dong-Hong-Li/net_retrofit_kit/blob/main/lib/src/network/net_options.dart)：

```dart
const NetOptions({
  required String baseUrl,
  Duration connectTimeout = const Duration(seconds: 60),
  Duration receiveTimeout = const Duration(seconds: 60),
  Duration sendTimeout = const Duration(seconds: 60),
  List<Interceptor>? interceptors,
});
```

| 字段 | 说明 |
|------|------|
| `baseUrl` | 所有请求的基础 URL（生成代码使用 baseUrl + path）。 |
| `connectTimeout` | 连接超时。 |
| `receiveTimeout` | 接收超时。 |
| `sendTimeout` | 发送超时。 |
| `interceptors` | Dio 拦截器；在创建 Dio 时会在默认日志拦截器之后按序添加。 |

---

## 设置默认 Client（NetRequest.options）

在**任意请求之前**必须设置 `NetRequest.options`，一般在 `main()` 中：

```dart
void main() {
  NetRequest.options = const NetOptions(
    baseUrl: 'https://api.example.com',
    connectTimeout: Duration(seconds: 30),
    receiveTimeout: Duration(seconds: 30),
    sendTimeout: Duration(seconds: 30),
    interceptors: [/* 可选 */],
  );
  runApp(const MyApp());
}
```

设置 `options` 会通过 [NetRequest.createDio](https://github.com/Dong-Hong-Li/net_retrofit_kit/blob/main/lib/src/network/net_request.dart) 创建新的 Dio，并作为**默认** Client 使用。未设置时，访问 `NetRequest.options` 或发起请求会抛出 `StateError`。

## createDio 与自定义 Dio

[NetRequest.createDio(NetOptions options)](https://github.com/Dong-Hong-Li/net_retrofit_kit/blob/main/lib/src/network/net_request.dart) 会创建包含以下内容的 Dio：

- 由 `NetOptions` 得到的 `BaseOptions`
- 包内默认的日志拦截器
- `NetOptions.interceptors` 中的拦截器

可以用 `createDio` 为**具名** Client（如 upload）单独建一个 Dio，再包成 [INetClient](https://github.com/Dong-Hong-Li/net_retrofit_kit/blob/main/lib/src/network/inet_client.dart) 或 [DefaultNetClient](https://github.com/Dong-Hong-Li/net_retrofit_kit/blob/main/lib/src/network/default/default_net_client.dart) 注册：

```dart
final uploadDio = NetRequest.createDio(const NetOptions(
  baseUrl: 'https://upload.example.com',
  receiveTimeout: Duration(seconds: 120),
));
NetRequest.setClient('upload', DefaultNetClient(uploadDio));
```

也可以完全自己创建 Dio（不同 baseUrl、拦截器等）再交给自定义 Client；见 [docs/04-多Client.md](https://github.com/Dong-Hong-Li/net_retrofit_kit/blob/main/docs/04-多Client.md)。

## 拦截器

- **默认**：设置 `NetRequest.options` 或调用 `NetRequest.createDio` 时，会先加入包内的 [HttpLoggingInterceptor](https://github.com/Dong-Hong-Li/net_retrofit_kit/blob/main/lib/src/network/http_logging_interceptor.dart)。
- **业务拦截器**：通过 `NetOptions.interceptors` 传入，会按顺序追加在日志拦截器之后。
- **运行时**：Dio 实例创建后，可调用 `NetRequest.addInterceptor(interceptor)` 或 `NetRequest.addInterceptors(list)` 继续追加。

---

## 使用已有 Dio（如单测）

单测或已有配置好的 Dio 时：

```dart
NetRequest.dioInstance = myDio;  // 或 NetRequest.use(myDio)
```

若生成代码或其他逻辑会读 `NetRequest.options.baseUrl`，仍需先设置 `NetRequest.options`。单测中可先设一个占位 `NetOptions`，再通过 `dioInstance` 注入 Mock Dio。

---

## 小结

| 需求 | 做法 |
|------|------|
| 默认 baseUrl、超时 | 在发起请求前执行一次 `NetRequest.options = NetOptions(...)` |
| 额外拦截器 | 使用 `NetOptions.interceptors` 或 `NetRequest.addInterceptor` / `addInterceptors` |
| 为具名 Client 建 Dio | `NetRequest.createDio(NetOptions(...))` 后 `setClient(name, client)` |
| 替换 Dio（如单测） | `NetRequest.dioInstance = dio` |
