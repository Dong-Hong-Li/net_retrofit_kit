# 示例工程

**运行：**

```bash
cd example
flutter pub get
dart run build_runner build --delete-conflicting-outputs
flutter run
```

---

## 案例速查


| 案例        | 演示内容                                                                                        |
| --------- | ------------------------------------------------------------------------------------------- |
| 1 基础      | `@Get` / `@Post`、`@Body`、`@StreamResponse`                                                  |
| 2 User    | `@QueryKey`、`@Path`、`@Header`、`@Query()`                                                    |
| 3 Article | `@Post` / `@Put` / `@Delete`、`@Body`、`@Path('id')`                                          |
| 4 上传      | **自定义 Client**：`@NetApi(client: 'upload')`、`UploadNetClient`、`ContentType.formData`、`@Part` |
| 5 Nested  | `@DataPath('result')`                                                                       |


请求均发往 `https://httpbin.org`（在 `main.dart` 配置）。

---

## 主要文件

- `main.dart` — 配置、`setClient('upload', UploadNetClient(dio))`
- `network/upload_net_client.dart` — 自定义 `INetClient`
- `server/upload_api.dart` — `@NetApi(client: 'upload')`
- `server/demo_server.dart` — 流式、基础 API
- `server/user_api.dart`、`article_api.dart`、`nested_api.dart` — 其他案例
