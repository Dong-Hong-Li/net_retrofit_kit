# 配置

**速查：**

| 需求 | 做法 |
|------|------|
| 默认 baseUrl、超时 | 在任意请求前执行一次 `NetRequest.options = NetOptions(...)` |
| 额外拦截器 | `NetOptions.interceptors` 或 `NetRequest.addInterceptor` / `addInterceptors` |
| 为具名 Client 建 Dio | `NetRequest.createDio(NetOptions(...))` 后 `setClient(name, client)` |
| 替换 Dio（如单测） | `NetRequest.dioInstance = myDio` |

---

## NetOptions

```dart
const NetOptions({
  required String baseUrl,
  Duration connectTimeout = Duration(seconds: 60),
  Duration receiveTimeout = Duration(seconds: 60),
  Duration sendTimeout = Duration(seconds: 60),
  List<Interceptor>? interceptors,
});
```

| 字段 | 用途 |
|------|------|
| `baseUrl` | 基础 URL；生成代码用 baseUrl + path。 |
| `connectTimeout` / `receiveTimeout` / `sendTimeout` | 超时。 |
| `interceptors` | 在默认日志拦截器之后添加。 |

---

## 设置默认（必须）

```dart
void main() {
  NetRequest.options = const NetOptions(
    baseUrl: 'https://api.example.com',
    connectTimeout: Duration(seconds: 30),
    receiveTimeout: Duration(seconds: 30),
    sendTimeout: Duration(seconds: 30),
  );
  runApp(const MyApp());
}
```

必须在首次请求前执行。

---

## createDio（具名 Client）

```dart
final uploadDio = NetRequest.createDio(const NetOptions(
  baseUrl: 'https://upload.example.com',
  receiveTimeout: Duration(seconds: 120),
));
NetRequest.setClient('upload', YourNetClient(uploadDio));  // 需自行实现 INetClient，示例工程有可参考实现
```

---

## 拦截器

- 默认：创建 Dio 时会加日志拦截器。
- 自定义：`NetOptions.interceptors` 或之后 `NetRequest.addInterceptor(interceptor)`。

---

## 单测：注入 Dio

```dart
NetRequest.dioInstance = myDio;  // 或 NetRequest.use(myDio)
```

若代码会读 `NetRequest.options.baseUrl`，需先设置 `NetRequest.options`。
